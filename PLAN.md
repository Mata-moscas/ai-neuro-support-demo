## Этап 1. Vector Store со сценариями
**Цель:** Агент может по запросу клиента найти подходящий сценарий обработки.

- [ ] Проанализировать `Instructions.json` — структура, качество данных, вложенность

- [ ] Написать скрипт индексации: разбить 114 сценариев на чанки, сгенерировать эмбеддинги, загрузить в Vector Store
- [ ] Написать модуль поиска: по тексту запроса клиента возвращает top-K релевантных сценариев
- [ ] Протестировать поиск на примерах из `Описание сценария.docx`

## Этап 2. MCP-серверы для эмуляции внутренних систем
**Цель:** Агент может получать данные о клиенте через MCP-вызовы.

На основе примеров из `Описание сценария.docx` создать MCP-серверы, эмулирующие:

- [ ] **customer_meta** — базовая мета клиента: тариф, ФИО, дата следующего списания, базовая цена
- [ ] **billing** — баланс, статус, кредитный лимит, блокировки; история списаний за 30 дней
- [ ] **incidents** — инциденты на сети (плановые работы, аварии) для абонента
- [ ] **subscriptions** — подписки клиента (контентные и сервисные)

Каждый сервер:
- Использует FastMCP (по аналогии с `weather-mcp`)
- Возвращает мок-данные (захардкоженные примеры из docx)
- Упакован в Docker
- Имеет описания инструментов (docstrings) для LLM

## Этап 3. Основной агент
**Цель:** Агент принимает вопрос, классифицирует, собирает данные, формирует ответ.

- [ ] Выбрать LLM-фреймворк (варианты: Claude API + tool use, LangChain, LangGraph)

- [ ] Реализовать маршрутизатор — определяет, какие MCP-серверы нужно вызвать для данного сценария
- [ ] Реализовать генератор ответа — на основе сценария и собранных данных формирует ответ
- [ ] Поддержка двух форматов вывода:

  - JSON для передачи в НейроСаппорт
- [ ] Написать системный промпт агента

## Этап 4. API и интеграция
**Цель:** Агент доступен через REST API.

- [ ] FastAPI-приложение с эндпоинтами:
  - `POST /ask` — основной запрос (вопрос клиента → ответ агента)
  - `POST /ask/{conversation_id}` — продолжение диалога (уточняющие вопросы)
- [ ] Pydantic-схемы для запросов/ответов
- [ ] Docker Compose для оркестрации всех сервисов (агент + MCP-серверы)
