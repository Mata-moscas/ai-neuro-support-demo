# AI Neuro Support Demo

Прототип агента-ассистента для поддержки клиентов телеком-оператора (МТС).
Агент автоматически классифицирует обращение, находит нужный сценарий, собирает данные о клиенте из внутренних систем и формирует ответ.

         ▼

## Ключевые требования

- Агент верно классифицирует сценарий либо явно сообщает о неуверенности
- Агент ведёт множество сценариев самостоятельно
- Агент находит информацию о клиенте в различных внутренних системах (через MCP)
- Ответ в формате текста (черновик) или JSON (для внешних систем)
- Поддержка уточняющих вопросов, если данных недостаточно
- Обработка мульти-интентных запросов
- Возможность диалога оператора с агентом для корректировки ответа
